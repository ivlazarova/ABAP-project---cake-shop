# ABAP
This program is a simple cake shop managment system written in ABAP. It allows browsing a cake catalog, placing orders with custom options and managing customer records. 

# Features:
1. Catalog view - browse all 20 available cakes with name, type, description, price and stock quantity
2. Order placement - you can order one or more cakes by entering their IDs, separated by commas
3. Portion size - you can choose cake pieces - 8, 16 or whole cake
4. Custom adds - you can add personal inscription, candles and fountain candles to the order
5. Automatic discounts - discount is applied based on the number of cakes for the order
6. Order cancellation - this is a option to cancel the whole order

# Pricing 
* Inscription - 5.00 EUR
* Candles, per piece - 1.20 EUR
* Fountain candle, per piece - 2.50 EUR
* 8 piece cake / whole cake - base price
* 16 piece cake - base price*2

# Discount:
* <3 cakes - 0%
* >3 or <5 cakes - 5%
* >5 or <15 cakes - 10%
* >15 cakes - 15%
